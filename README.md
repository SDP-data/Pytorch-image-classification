# PyTorch Image Classification

Image classification project using PyTorch and a Convolutional Neural Network (CNN) trained on the CIFAR-10 dataset.

## Project Overview

This project implements a CNN for image classification using PyTorch. The model is trained and evaluated on CIFAR-10, a dataset containing 60,000 images across 10 classes.

## Dataset

CIFAR-10 classes:

- plane
- car
- bird
- cat
- deer
- dog
- frog
- horse
- ship
- truck

## Tech Stack
